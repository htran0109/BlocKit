FAQ

BlocKIT blocks out your distractions so that you can stay focused on completing your tasks. You will be able to create tasks with customized blocked websites, so when you start the task, those distracting websites will be blocked until either the timer runs out or you finish your task.

The Create Task button on the home screen will take you to another screen to input your task’s name, due date, time duration, notes, and customize the blocked websites. The “Create!” button will finish creating the task.

The Task page will contain a list of your created tasks. There, you can select a task to run.

Running a task will block the websites you customized for that task for the duration of the time you set for that specific task.

You can manually stop the timer when you finish your task, which will then unblock your websites.

You can go to the Settings page to customize the default blocked websites.
